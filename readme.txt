-------------------------------------Project Code 21 . GPS based Payroll -----------------------------------------


Instructions : 

1. This project uses lampp stack

2. Install xammp/lamp on your machine 

3. Put the payroll folder in htdocs 

4. Change login.php login credentials for your lampp setup

5. Run create.php from the automate directory to create the database named payroll. This file also creates tables named auth and hr_table.

6. Go to your localhost/Payroll. Login with username as admin and password pass to access the admin account.
